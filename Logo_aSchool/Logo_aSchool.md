# aSchool — Logo Horizontal (Version Finale 2026)

## 1. Description
Le logo horizontal combine :
- le **a ouvert** rouge bordeaux (#7A1F2A)
- la **trame IA géométrique** bleue azur (#0096D6)
- le **livre double ligne** bleu azur
- le mot **School** en bleu azur, avec un **S majuscule plus haut que le a** (+10 %)
- la baseline : **Générateur d’activités pédagogiques**

## 2. Proportions finales
- Le **S** = **hauteur du a +10 %**
- Le **a** est réduit pour ne pas dominer la majuscule
- Alignement horizontal parfait
- Baseline centrée sous “aSchool”

## 3. Usages
- Header web
- Emails
- Présentations
- Documents officiels

## 4. Formats fournis
- SVG (vectoriel)
- PNG (fond transparent)
- PDF (print)

## 5. Zones de protection
- Zone = hauteur du “a”
- Aucun élément ne doit entrer dans cette zone

## 6. Tailles minimales
- Digital : 120 px
- Print : 20 mm

## 7. Interdits
- Modifier les couleurs
- Déformer les proportions
- Ajouter ombres / glow / dégradés
- Coller le livre au “a”
