# Installer aSchool sur iPhone (iOS)

> Fonctionne uniquement avec **Safari** — pas Chrome, pas Firefox.

## Étapes

1. Ouvrez **aschool.fr** dans Safari et connectez-vous.
2. Appuyez sur l'icône **Partage** (📤) en bas au centre de Safari.
3. Faites défiler la liste vers le bas et appuyez sur **« Sur l'écran d'accueil »**.
4. Vérifiez que le nom affiché est **aSchool**, puis appuyez sur **Ajouter**.
5. L'icône aSchool apparaît sur votre écran d'accueil. Ouvrez-la — l'app se lance en plein écran, sans barre Safari.

## Notes

- Une bannière bordeaux apparaît automatiquement sur le dashboard après 5 secondes pour vous guider.
- Une fois installée, aSchool fonctionne même avec une connexion faible — l'interface se charge instantanément.
- Si la bannière a été fermée, retrouvez cette procédure dans **Aide → Installer aSchool sur iPhone**.
