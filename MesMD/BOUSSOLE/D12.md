## Checklist de validation (flow utilisateur complet)

À cocher en testant comme un prof réel sur Edge desktop + mobile :

### Entrée utilisateur
- [ ] Saisie texte directe → génération OK
- [ ] OCR image JPG/PNG → texte extrait correct → génération OK
- [ ] Dictée vocale (Groq Whisper batch, [D15](D15.md)) → texte injecté → génération OK

### Génération
- [ ] Tous types d'activités testés (au moins 1 par matière × 4 types principaux)
- [ ] Niveau changé via profil → impact visible dans le résultat
- [ ] Avec/sans correction → comportement attendu
- [ ] Few-shot : après 3 saves même type, adaptation observable

### Résultat
- [ ] Markdown rendu correctement (titres, listes, gras)
- [ ] LaTeX maths : `$...$` et `$$...$$` rendus (KaTeX/MathJax)
- [ ] Pas de débord visuel, pas de texte coupé

### Persistance + historique
- [ ] Sauvegarde auto fonctionne
- [ ] `MesActivites` : liste affichée, tri OK, filtre profil auto OK
- [ ] Détail activité : affichage complet, bouton supprimer avec confirm
- [ ] Suppression : confirm modal + retrait visible immédiat

### Export
- [ ] DOCX : ouvre dans Word, mise en forme propre, footer `aschool.fr`
- [ ] Mailto : corps + signature `aSchool — aschool.fr` intacts
- [ ] Copier : texte propre dans presse-papier

### Erreurs (modal showError partout)
- [ ] 413 payload trop gros → modal
- [ ] 503 Groq down → modal + fallback chain (D04)
- [ ] Timeout → modal
- [ ] Texte source vide → modal
- [ ] Pas une seule erreur en `<div>` inline

### Mobile (`window.innerWidth < 768`)
- [ ] Sidebar repliée par défaut
- [ ] Header : tagline masquée, matière sous nom
- [ ] Layout 1 colonne, pas de débord
- [ ] Boutons hover-only → toujours visibles

## Méthode

1. **Audit** — 1 session : un prof simule le flow complet, je note chaque aspérité dans la checklist
2. **Fix** — 1-N sessions : chaque case rouge → bug ouvert, corrigé, validé. Chaque correction non triviale → traçage dans BOUSSOLE comme sous-D si elle dépasse 30 min
3. **Validation pilotes** — 3 profs réels testent et confirment "ça marche sans bug bloquant"
