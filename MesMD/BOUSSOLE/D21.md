# D21 — Mémo flash (format révision rapide)

**Statut :** ☐ à faire — après le moteur visuel [D20](D20.md)
**Item sommaire :** [33](../TABLEAU-DE-BORD.md#item-33)

## Objectif

Format ultra-condensé recto/verso, idéal pour la **révision rapide avant un contrôle**. Distinct de la fiche de révision classique (plus longue, plus détaillée).

## Ce qui existe / ce qui manque

**Existe :** type `fiche_revision` dans les 12 matières — sortie texte structuré + export Word. Un seul format.
**Manque :** le format mémo flash, ultra-condensé.

## Implémentation

- 1 nouveau type dans `MesMD/MATRICE_ACTIVITES_ASCHOOL.md`
- Nouveau prompt par matière concernée
- Même moteur de génération, même endpoint `/api/generate`

## Renvoi

Une fois le **moteur visuel** ([D20](D20.md)) livré, le mémo flash peut inclure une **version carte mentale visuelle**. Renvoi, pas dépendance bloquante : le format texte fonctionne seul.
