# D22 — Supports de créativité élève

**Statut :** ☐ à faire — canevas BD après le moteur visuel [D20](D20.md)
**Item sommaire :** [34](../TABLEAU-DE-BORD.md#item-34)

## Objectif

Étendre les supports de créativité élève au-delà du Français — multi-matières.

## Ce qui existe / ce qui manque

**Existe :** en Français uniquement — types « continuer un texte », « décrire un personnage », « imaginer la suite d'une scène », « texte poétique ». Une seule matière couverte.

**Manque :**
1. **Amorces d'écriture** — phrase d'ouverture fournie, l'élève continue. Type dédié, multi-matières.
2. **Situations-problèmes ouvertes** — un obstacle, une contrainte, une décision à prendre. Toutes matières.
3. **Défis structurés** — contraintes formelles créatives (« écrivez en 5 phrases sans le mot *important* », « résolvez sans utiliser la multiplication »).
4. **Canevas BD** — structure narrative en cases. Nécessite le **moteur visuel** ([D20](D20.md)) pour être vraiment utile, sinon c'est du texte décrivant une BD.

## Implémentation

- 3-4 nouveaux types dans `MesMD/MATRICE_ACTIVITES_ASCHOOL.md`
- Prompts multi-matières dans `src/prompts.py`
- Même moteur de génération.
