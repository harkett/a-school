## Ce que c'était

BTS CIEL Option A avait été traité **AVANT les réformes**, à l'ancienne méthode : référentiel déposé + fiche Python de réglages + chunks vectorisés. Cet artefact risquait d'**entrer en conflit** avec la nouvelle procédure data-driven quand on traiterait le cycle BTS.

## Décision prise — supprimer le BTS fonctionnel, garder la structure

Plutôt que réconcilier, on a **retiré tout le contenu BTS** pour le régénérer plus tard via la procédure standard (celle qu'on met au point sur la crèche). Retiré le 08/07/2026 :

- **Code** : la fiche Python du BTS CIEL (supprimée) + son entrée dans le registre `FICHES` + les 2 valeurs par défaut périmées de `pgvector_store.py`.
- **Disque** : le dossier du référentiel BTS (PDF officiel conservé à part par Harketti).
- **Base** : le référentiel BTS (sa collection) + ses **236 chunks** + les liens matières (0 de toute façon). Prouvé sur le miroir `aschool_creche_miroir` ; SQL rejoué sur la vraie base par Harketti.
- **Tests** : `test_rag_ciel.py` (bout-en-bout BTS) supprimé ; `test_exemple_referentiel.py` reciblé sur la crèche.

**Conservé volontairement** : le **cycle « BTS » (id 8) + ses 17 niveaux** (dont CIEL A=42, CIEL B=43). C'est l'échafaudage (décision humaine, snapshot `outils_bdd/cycles_niveaux.json`), pas l'artefact — même statut que les niveaux crèche. Le BTS reste « déclaré », vide de contenu, prêt à être régénéré.

## Leçons à réappliquer quand on régénérera le BTS

- **Casse du niveau.** L'ancienne fiche posait `NIVEAU = "BTS CIEL option A"` (« option » minuscule) ≠ base `niveaux.nom = "BTS CIEL Option A"` (majuscule) → tout appariement par chaîne exacte échouait en silence. À la régénération : **aligner exactement** le libellé.
- **Fiche Python vs data-driven.** L'ancienne fiche était le « code par cas » que la doctrine `couple_data_driven` veut réduire. La question « fiche Python par spécialité vs registre data-driven » reste ouverte → [D57](D57.md).
- **Matières.** Les 9 matières CIEL du seed supprimé ne sont **plus** en base ; elles seront **extraites du référentiel** à la régénération, jamais saisies à la main.
