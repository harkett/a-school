# D27 — Sortie séquence en JSON structuré (rendu pro)

**Statut :** ☐ à faire — après l'affinage interactif [D07](D07.md)
**Item sommaire :** [38](../TABLEAU-DE-BORD.md#item-38)

## Objectif

Passer du markdown brut affiché dans une `<div whiteSpace: pre-wrap>` à un **rendu React structuré** : cartes par phase, badges durée, code couleur par type d'organisation.

## Ce qui existe / ce qui manque

**Existe :** le Générateur de séquences renvoie du markdown brut — pas esthétique, pas exploitable structurellement.

**Manque :**
- Sortie JSON typée : `titre, matiere, niveau, duree_totale, phases: [{nom, duree, objectif, deroulement, organisation}]`
- Rendu React dédié avec cartes par phase, badges durée, code couleur par type d'organisation (individuel / binôme / groupe / collectif)

## Implémentation

1. Modifier le prompt du Générateur de séquences → `response_format: json_object`
2. Composant `SequenceRenderer.jsx` qui mappe le JSON sur des cartes
3. **Migration douce** : garder le markdown en fallback si le parsing JSON échoue

## Bénéfices second-ordre

- Facilite l'affinage interactif ([D07](D07.md)) : l'affinage peut cibler une phase précise par index
- Facilite le versioning ([D26](D26.md)) : versions diff par phase
