# D18 — Mode expérience prof (T1 / confirmé / expert)

**Statut :** ☐ à faire
**Item sommaire :** [29](../TABLEAU-DE-BORD.md#item-29)

## Objectif

Adapter le **niveau de détail** des générations selon l'expérience du prof.

## Ce qui existe / ce qui manque

**Existe :** rien — tous les profs reçoivent le même niveau de détail.
**Manque :**
- un champ `experience` dans la table `User` (valeurs : T1 / confirmé / expert)
- le champ dans `MonProfil.jsx`
- une variable injectée dans **tous les prompts existants**

## Modificateurs de prompt

| Profil | Modificateur |
|---|---|
| **T1** | scripts d'oralisation, conseils de gestion de classe, étapes très détaillées |
| **Expert** | format condensé, pas d'explication des bases, raccourcis pédagogiques |

## Implémentation

Pas de nouvelle page, ni composant, ni endpoint. **Un champ BDD + une variable prompt** qui traverse toutes les features — plus on attend, plus il manque partout.
