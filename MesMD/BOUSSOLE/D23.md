# D23 — Différenciation DYS / FLE / approfondissement

**Statut :** ☐ à faire
**Item sommaire :** [30](../BACKLOG.md#item-30)

## Objectif

Produire **3 variantes pédagogiques** d'une activité déjà générée pour s'adapter aux profils d'élèves. Adapte le **fond** — à distinguer du few-shot (livré 30/04/2026) qui adapte la **forme** au style du prof, pas la pédagogie.

## Ce qui existe / ce qui manque

**Existe :** few-shot adaptation au style du prof — adapte la forme, pas le fond.
**Manque :** les 3 variantes pédagogiques.

## Les 3 variantes

| Version | Caractéristiques |
|---|---|
| **DYS** | Syntaxe épurée, phrases courtes, structure simple, vocabulaire non ambigu — police OpenDyslexic côté frontend uniquement (CSS) |
| **FLE / allophones** | Vocabulaire simplifié, contextes culturels neutres, langage quotidien |
| **Approfondissement** | Nuances ajoutées, complexification, ouverture vers notions connexes |

## Implémentation

- 3 boutons dans `ZoneResultat.jsx` après génération (« Version DYS », « Version FLE », « Version approfondissement »)
- 3 modificateurs de prompt côté backend injectés dans l'endpoint existant `/api/generate`
- **Pas de nouveau backend, pas de nouvelle page.**
