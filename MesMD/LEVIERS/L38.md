## Objectif

Passer du markdown brut affiché dans une `<div whiteSpace: pre-wrap>` à un **rendu React structuré** : cartes par phase, badges durée, code couleur par type d'organisation.

---

## Ce qui existe / ce qui manque

**Existe :** L1 renvoie du markdown brut — pas esthétique, pas exploitable structurellement.

**Manque :**
- Sortie JSON typée : `titre, matiere, niveau, duree_totale, phases: [{nom, duree, objectif, deroulement, organisation}]`
- Rendu React dédié avec cartes par phase, badges durée, code couleur par type d'organisation (individuel / binôme / groupe / collectif)

---

## Implémentation

1. Modifier prompt L1 → `response_format: json_object`
2. Composant `SequenceRenderer.jsx` qui mappe le JSON sur des cartes
3. **Migration douce** : garder le markdown en fallback si parsing JSON échoue
