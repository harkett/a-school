## Objectif

Adapter le niveau de détail des générations selon l'expérience du prof.

---

## Ce qui existe / ce qui manque

**Existe :** rien — tous les profs reçoivent le même niveau de détail dans les générations.

**Manque :**
- 1 champ `experience` dans la table `User` (valeurs : T1 / confirmé / expert)
- Champ dans `MonProfil.jsx`
- Variable injectée dans tous les prompts existants

---

## Modificateurs de prompt

| Profil | Modificateur |
|---|---|
| **T1** | Scripts d'oralisation, conseils de gestion de classe, étapes très détaillées |
| **Expert** | Format condensé, pas d'explication des bases, raccourcis pédagogiques |

---

## Implémentation

**Pas de nouvelle page, pas de nouveau composant, pas de nouveau endpoint.**

Un champ BDD + une variable prompt qui traverse toutes les features — plus on attend, plus il manque partout.
