# Mémo Markdown — Formatage rapide

> **Document personnel — aide-mémoire syntaxe Markdown. Non lié au développement de A-SCHOOL.**
>
> Ce document contient la syntaxe Markdown de référence : titres (H1/H2/H3), mise en forme (gras, italique, barré), listes (puces, numérotées, cases à cocher), citations, séparateurs, tableaux, liens, blocs de code (inline et multi-lignes), HTML inline pour couleurs et styles.
>
> **À conserver ici pour un accès rapide lors de la rédaction des documents `.md` du projet.**

---

## Titres

```
# Titre principal (H1)
## Titre secondaire (H2)
### Titre tertiaire (H3)
```

---

## Mise en forme du texte

```
**gras**
*italique*
***gras et italique***
~~barré~~
```

---

## Listes

**À puces**
```
- élément
- élément
  - sous-élément (2 espaces avant le tiret)
```

**Numérotées**
```
1. premier
2. deuxième
3. troisième
```

**Cases à cocher**
```
- [ ] à faire
- [x] fait
```

---

## Citations et notes

```
> Texte mis en avant — apparaît encadré
```

---

## Séparateur

```
---
```

---

## Tableaux

```
| Colonne 1 | Colonne 2 | Colonne 3 |
|---|---|---|
| valeur | valeur | valeur |
| valeur | valeur | valeur |
```

---

## Liens

```
[texte affiché](https://url.com)
```

---

## Code

**Inline (dans une phrase)**
```
Utilisez la commande `git pull`
```

**Bloc de code**
````
```
votre code ici
```
````

---

## HTML inline (pour la couleur ou le style)

```html
<span style="color:#A63045">texte coloré</span>
<span style="font-size:11px">texte plus petit</span>
<strong>gras HTML</strong>
```

---

## Récapitulatif visuel

| Syntaxe | Résultat |
|---|---|
| `# Titre` | Gros titre |
| `## Titre` | Titre moyen |
| `### Titre` | Petit titre |
| `**mot**` | **gras** |
| `*mot*` | *italique* |
| `- élément` | puce |
| `1. élément` | numéroté |
| `> texte` | citation |
| `---` | ligne de séparation |
| `` `code` `` | code inline |
